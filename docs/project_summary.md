# Project Summary

## Project Title
Live Stock Market Intelligence Dashboard

## Project Overview
This project is an API-connected Power BI dashboard designed to analyze stock market performance across selected technology companies. The dashboard allows users to review stock price trends, trading volume, daily performance, risk indicators, intraday returns, and company-level comparisons.

## Business Problem
Investors, analysts, and business users need a simple and interactive way to monitor stock market activity and compare company performance. Raw stock market data can be difficult to interpret without visual summaries, trend analysis, and risk indicators.

This project solves that problem by creating an interactive Power BI dashboard that converts daily stock market data into clear business insights.

## Dashboard Objective
The objective of this dashboard is to provide a refreshable market intelligence solution where users can:

- Analyze individual stock performance
- Review price and volume trends
- Evaluate daily price range and intraday return
- Compare multiple technology stocks side by side
- Use slicers to filter by ticker and rolling date window

## Companies Analyzed
The dashboard includes the following stock tickers:

- AAPL
- AMZN
- MSFT
- NVDA
- TSLA

## Tools Used
- Power BI
- Power Query
- DAX
- Yahoo Finance Web API
- GitHub

## Dashboard Pages
1. Executive Overview
2. Stock Performance Analysis
3. Risk and Volatility Analysis
4. Company Comparison

## Project Outcome
The final dashboard provides a professional, interactive stock market analysis solution that supports both individual stock review and multi-company comparison.
